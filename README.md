# C3 Autos Blindados - Projeto Web

Projeto acadêmico desenvolvido para a disciplina de Programação para Internet.

## Sobre o projeto
Desenvolvimento de um website institucional para uma empresa fictícia de veículos blindados.

## Estrutura
- Home
- Veículos
- Serviços
- Sobre nós
- Contato

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript
